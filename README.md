# Primal Fear Dex

A comprehensive companion app for the Primal Fear mod in ARK: Survival Evolved, inspired by the functionality and structure of Dododex.

## Features

- **Complete Database**: All creatures, items, structures, resources, and bosses from Primal Fear mod
- **Advanced Calculators**: Taming, breeding, stats, and crafting calculators
- **Smart Search**: Instant search across all data types with filtering
- **Tier System**: Full progression through Alpha, Beta, Gamma, Delta, Epsilon, and Omega tiers
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Offline Support**: Cached data for offline browsing
- **Community Features**: Tips, strategies, and user contributions

## Technology Stack

- **Frontend**: Pure HTML, CSS, and JavaScript (no frameworks)
- **Architecture**: Single Page Application (SPA)
- **Data**: JSON-based with automated loading and caching
- **Hosting**: GitHub Pages (static hosting)
- **Performance**: Lazy loading, caching, and optimization
